# Interview DSA Patterns (Must Recognize Instantly)

## Sliding Window

Used when:

- Subarray / substring
- Contiguous elements
  Examples:
- Longest substring
- Min window substring

## Two Pointers

Used when:

- Sorted array
- Pair / triplet
  Examples:
- Two Sum
- 3Sum

## Hashing

Used when:

- Frequency
- Fast lookup
  Examples:
- Anagrams
- Subarray sum

## Stack

Used when:

- Next greater/smaller
- Valid parentheses

## BFS

Used when:

- Shortest path (unweighted)
- Level-wise traversal

## DFS

Used when:

- Cycle detection
- Tree recursion

## DP

Used when:

- Choices + optimal result
- Overlapping subproblems

## Binary Search on Answer

Used when:

- Min / max possible value
- Feasibility check
