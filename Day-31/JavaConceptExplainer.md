# Java Interview Explainers

## What is HashMap?

A HashMap stores key-value pairs and allows O(1) average time for get/put.

## Why HashMap is not thread-safe?

Because multiple threads can modify it without synchronization.

## Difference between String and StringBuilder?

String is immutable, StringBuilder is mutable and faster in loops.

## What is JVM?

JVM executes Java bytecode and manages memory, GC, and threads.
