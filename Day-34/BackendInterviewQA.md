# Backend Interview Q&A

## What is MVC?

- Model: Business logic
- View: UI
- Controller: Handles requests

## What is API latency?

- Time taken to process a request

## What is pagination?

- Breaking large responses into pages

## What is caching?

- Storing frequently used data temporarily

## How do you secure APIs?

- Authentication (JWT/OAuth)
- Authorization
- HTTPS
